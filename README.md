# n8n PM Workflows

This repository is for sharing reusable n8n workflows tailored to project management (PM) processes.

## Purpose

- Provide a central place for PM workflow examples and documentation.

## Structure

- `workflows/`: Exported n8n workflow files in JSON format.
  - `SRanalysis.json`: Template for service request analysis

## How to use

1. Download or clone this repo.
2. In n8n, go to `Import` and select a JSON workflow file from the `workflows/` folder.
3. Customize the nodes and credentials for your environment.
